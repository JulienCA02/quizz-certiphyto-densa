# Quiz d'entraînement Certiphyto

Ce dépôt contient un quiz interactif d'entraînement au **Certiphyto**.  
Il permet de s’exercer à partir d’un pool de questions aléatoires et d’obtenir un score en fin de test.

---

## Fonctionnement

- x questions disponibles dans le fichier (en cours d’enrichissement) `questions.json`
- les questions sont tirées **aléatoirement** à chaque session
- Les questions, leur nombre et le seuil de réussite sont adaptés à la **catégorie de Certiphyto**
